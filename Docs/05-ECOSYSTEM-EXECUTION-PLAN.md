## Phase 2: Product Requirements Document (PRD)

### 1. Consumer App (Refining Current Scope)
*   **Core:** AI Body Scanning & parametric 3D modeling.
*   **New Feature: Family Size Wallet:** Manage profiles for family members.
*   **New Feature: Virtual Wardrobe:** Re-order previous cuts/styles.

### 2. Partner Portals
*   **Tailor Dashboard:**
    *   Receive "Body Information Modeling" (BIM) data.
    *   Capacity management tools.
*   **Fabric Supplier Hub:**
    *   Real-time inventory system.
    *   Upload fabric stock visible to customers/tailors.

### 3. B2B Corporate Dashboard
*   **Features:**
    *   Employee bulk upload.
    *   Self-service measurement links.
    *   Unified billing.
    *   Annual renewal automation.

## Phase 3: Copywriting & Brand Messaging

### 1. Sustainability Narrative (ESG Focus)
*   Shift from "Buying a Thobe" to "Joining a Sustainable Fashion Revolution".
*   Keywords: "Zero-Waste Manufacturing", "Eco-Conscious Tailoring".
*   Focus on reduced returns and carbon footprint.

### 2. Empowerment Narrative
*   "Join Us" pages focus on digital infrastructure for local tailors/SMEs.
*   Enable global competition through Digital Transformation.

## Phase 4: Technical Strategy

### 1. Tech Stack Recommendation
*   **Backend:** Python/TensorFlow for Computer Vision & AI processing. Node.js/Go for scalable API services.
*   **Frontend:** React (Web), Flutter (Mobile App).
*   **Database:** Unified schema linking Fabric SKU -> Order -> Task.

### 2. API Strategy ("Qeyafa Connect")
*   SaaS model for third-party retailers to use the measurement engine.
